## A portfolio blog should still feel personal

The article page should not look like a generic docs site. It should carry the same spacing, typography, and tone as the rest of the portfolio.

## Layout decisions that help

I like using:

- A centered reading column
- Strong page title spacing
- One clear hero image area
- Soft cards for code and supporting content

## What I avoid

Too many side widgets can make the article feel fractured. If the goal is reading, the main column should stay in control.

## Keep the skeleton simple

A calm editorial structure is easier to extend later than a layout built around decorative extras.
